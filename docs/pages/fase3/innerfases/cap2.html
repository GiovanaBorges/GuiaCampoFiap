Mysql

Feito por : Arthur
Capítulo 2

MySQL é um sistema de gerenciamento de banco de dados relacional de código aberto, amplamente utilizado em aplicações web. Ele permite a criação e gerenciamento de bancos de dados, assim como a execução de consultas para recuperação e manipulação de informações armazenadas. O MySQL é conhecido por sua facilidade de uso e robustez, tornando-o uma escolha popular para desenvolvedores e empresas.

Ilustração da linguagem de banco de dados MYSQL

Beleza mas onde ele é usado
O MySQL é utilizado para armazenar, organizar e recuperar informações em um banco de dados. Ele funciona criando tabelas para armazenar informações específicas, como nomes de usuários, endereços de e-mail e produtos em uma loja virtual, por exemplo. O MySQL também permite que você execute consultas complexas para recuperar e manipular esses dados de várias maneiras, como filtrar registros, calcular somas e contagens, e agrupar dados. Isso torna o MySQL uma ferramenta poderosa para aplicações que requerem armazenamento de dados e análise de informações.
O QUE É BANCO DE DADOS RELACIONAL VS NÃO RELACIONAL

Banco de dados relacional e não relacional são dois tipos diferentes de sistemas de gerenciamento de banco de dados.

Banco de dados relacional: Um banco de dados relacional é baseado em tabelas e relações entre elas. Cada tabela armazena informações específicas, como nomes de usuários ou detalhes de produtos, e as relações entre tabelas permitem que as informações sejam vinculadas e recuperadas de maneira organizada. O MySQL é um exemplo de RDBMS.

Em resumo, o banco de dados relacional é uma boa escolha quando você precisa de uma estrutura de dados organizada e bem definida, enquanto o NoSQL é mais flexível e escalável, e é uma boa opção para aplicações que lidam com grandes quantidades de dados não estruturados.

PRINCIPAIS COMANDOS MYSQL

As principais comandos disponíveis no MySQL incluem:

  • SELECT: usado para recuperar informações do banco de dados.
  • INSERT: usado para inserir novos dados em uma tabela.
  • UPDATE: usado para atualizar informações existentes no banco de dados.
  • DELETE: usado para excluir dados do banco de dados.
  • CREATE: usado para criar novas tabelas, índices, banco de dados, etc.
  • DROP: usado para excluir tabelas, índices, banco de dados, etc.
  • ALTER: usado para modificar a estrutura de tabelas existentes, como adicionar ou excluir colunas.
  • INDEX: usado para criar ou excluir índices em uma tabela para melhorar a velocidade de consulta.
  • JOIN: usado para combinar informações de duas ou mais tabelas.
  • GROUP BY: usado para agrupar resultados de uma consulta por uma ou mais colunas.

Esses são apenas alguns dos comandos disponíveis no MySQL. Existem muitos outros que permitem que você realize tarefas mais avançadas, como transações, subconsultas e geração de relatórios.

EXERCÍCIOS MYSQL
Aqui estão alguns exemplos de exercícios que você pode fazer para melhorar suas habilidades com o MySQL:
  • Crie um banco de dados para armazenar informações sobre seus livros favoritos. Cada livro deve ter um título, autor, data de publicação e número de páginas.
  • Inserir pelo menos 5 livros no banco de dados criado.
  • Recuperar todos os livros escritos por um determinado autor.
  • Atualize a data de publicação de um livro específico.
  • Exclua um livro específico do banco de dados.
  • Adicione uma nova coluna a tabela para armazenar a classificação do livro em uma escala de 1 a 5.
  • Recuperar todos os livros com uma classificação acima de 4.
  • Agrupe os livros por ano de publicação e calcule a média de classificação para cada ano.

Esses são apenas alguns exemplos, mas há muito mais que você pode fazer para melhorar suas habilidades com o MySQL. Experimente criar consultas mais complexas, trabalhar com transações e aprender sobre técnicas de otimização de consultas.
